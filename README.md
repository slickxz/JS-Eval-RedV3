# JS-Eval-RedV3

# PROBABLY BROKEN
A JS eval cog for Red V3

## Installation
1. Open your venv.
```bash
source [path] 
```
2. Install Js2Py
```bash
pip install js2py
```
### Now to Discord
3. Add the repo
```bash
[p]repo add jseval https://github.com/slickxz/JS-Eval-RedV3
``` 
4. Install the cog
```
[p]cog install jseval eval
```

5. Done!
## Usage

```python
[p]eval [JS eval code]
```
